What is PLCheat?
<close>


A customizable cheat sheet for programming languages. Not a teaching tool, but "I understand it in X, what was Y's syntax again?". Searching for a quickie regexp usage shouldn't require digging through tutorials and lengthy autogenerated documents.

Please note: This HEAVILY uses OpenAI's chatgpt engines. There will probably be errors, misunderstandings, and nuclear explosions. The purpose is still met, I believe - syntax is the primary issue, here.

I would like to thank highlightjs.org for their fantastic code highlighting!

I'm a weirdo, I like searching with partial keywords, often with just my left hand. e.g: arr sor will match sort array. This toggle enables hijacking Ctrl+F to focus advanced search box.

Note: This only searches comments. So arr sort would match # Array Sorting, but not array.sort() in an example. It will match topics, so regex atch would match Pattern Matching under topic Regular Expressions. I try and add aliases to most topics, so Regular Expressions should have aliases: rx and regexps

(More Info)

PLCheat higlights words that match. e.g: a search for atter will highlight the entirety of Pattern. Toggling this makes it only highlight the atter portion.
